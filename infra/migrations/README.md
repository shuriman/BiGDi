# infra/migrations

Каталог для SQL-скриптов и вспомогательных материалов, связанных с миграциями PostgreSQL, которые не управляются напрямую Prisma.

## Что хранить здесь
- Скрипты включения расширений/ролей.
- Хотфиксы и ручные миграции, требующие отдельного контроля.
- Описания процедур бэкапа и отката.

Основные миграции схемы будут генерироваться Prisma (`apps/api/prisma/migrations`). Этот каталог служит дополнением для инфраструктурных задач бд.
