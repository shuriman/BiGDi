# Архитектурные артефакты Zemo

Этот каталог содержит все ключевые документы по реархитектурированию Zemo:

- `overview.md` — высокий уровень и ключевые принципы.
- `c4-diagrams.md` — диаграммы контекста, контейнеров и компонентов.
- `er-diagram.md` — ER-диаграмма и стратегия миграции данных.
- `openapi.yaml` — черновик API-контрактов.
- `nfr.md` — нефункциональные требования и SLO.
- `migration-plan.md` — план перехода и отката.
- `repo-guide.md` — рекомендации по структуре репозитория и кодстайлу.
- `adr/` — принятые решения (ADR 0001–0006).
- `diagrams/` — исходники диаграмм (Mermaid/PlantUML) и экспортируемые материалы.

Документы обновляются по мере уточнения требований и прогресса проекта. При изменении архитектурных решений необходимо создавать новые ADR или обновлять существующие с указанием статуса.
